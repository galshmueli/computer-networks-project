# computer-networks-project
TCP client server chat and traffic :
פרויקט זה בוצע במסגרת קורס רשתות תקשורת מחשבים.

הפרויקט כולל מערכת צ’אט מבוססת שרת לקוח המשתמשת בפרוטוקול TCP,
וניתוח תעבורת הרשת של המערכת באמצעות Wireshark.

הרצת הפרויקט:
1. יש להריץ תחילה את קובץ server.py.
2. לאחר מכן יש להריץ קובץ client.py ממחשב אחד או יותר.
3. הלקוח מתחבר לשרת באמצעות כתובת ה-IP והפורט המוגדרים בקוד.
4. לאחר החיבור ניתן לשלוח הודעות ולקבל תגובה מהשרת.

במאגר זה מצורפים קבצי הקוד, קובצי Wireshark, קובץ CSV, מחברת Jupyter והדוח המסכם.
